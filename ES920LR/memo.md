# ES920LR

コマンドはCRLF

動作モード: 1=terminal, 2=processor

## 設定

- 帯域幅
	- 普段125kHz?
	- b 4
- 拡散率
	- 普段7
	- あんまデカくしてもクソ遅い
	- c 7
- チャンネル: 921.6MHz
	- 帯域幅125kHzじゃないと設定できない?: チャンネル番号5
	- d 5
- RSSI付与
	- p 1
- baudrate
	- r 5 -> 115200
- format
	- バイナリ
	- A 2
- save
	- w
